Tuesday, 17th November 2015

I received Mahiri micro:bit and put it into simple robot, creating micro:bot:

The robot is very simple, build using ready parts and some two-sided adhesive tape, without soldering or any special tools (except for a screwdriver to attach the wheels to the servos).

I got the servos to move by using the pin.write_analog(), which outputs PWM signal on the pin:

>>> import microbit

>>> microbit.pin0.set_analog_period(20) # 50Hz
>>> microbit.pin1.set_analog_period(20)

>>> microbit.pin0.write_analog(20)
>>> microbit.pin1.write_analog(122)

You can also read the sensor very easily:

>>> import microbit

>>> print(microbit.pin2.read_analog())

The next task is to actually write some obstacle-avoidance code and upload it into the board. Creating all the accounts and getting all the tools to work so far proves to be a little difficult.
